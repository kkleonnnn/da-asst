## 正文

### 一句话方法论

选图表不要凭感觉，而要**先问「我想表达哪一种数据关系」**，再去对应的图表族里挑。一张图最多承载三个维度比较合适（可借散点大小、颜色深浅多塞一两维，但别太多，否则一团乱、看不清）。按数据关系，常见图表可归成七类。

### 七类数据关系 → 对应图表

| 想表达的关系 | 常用图表 | 说明 / 边界 |
|---|---|---|
| **趋势型**：变量随时间或某连续自变量怎么变 | 折线图、面积图、堆叠面积图、堆叠占比面积图、拟合曲线图、阶梯图 | 自变量也可是离散时间点（用阶梯图）；不仅看「随时间」，也能看「随某指标」的趋势（如存活率随温度）。回归分析的展示天然属于此类 |
| **比较型**：分组后两组及以上的对比 | 柱状 / 条形图、瀑布图、雷达图、词云 | 柱 / 条最直观；瀑布图把绝对值与增减变化结合；雷达图比多维均衡度（围成面积≈总体水平）；词云用字号比词频 |
| **比例型**：整体与各部分的构成占比 | 饼图、环状图、堆叠图（面积 / 柱状） | 饼 / 环在极坐标下展示；堆叠图在常见的直角坐标下展示，更适合同时看占比和总量 |
| **分布型**：数据如何分布 | 散点图、气泡图、直方图、概率密度图、箱线图、热力图 | 直方图≠柱状图（见下方「易错点」）；箱线图是看 IQR、上下界、离群点的利器；气泡图比散点多一维（气泡大小） |
| **区间型**：同一维度上不同分区的差异 | 进度条、仪表盘 | 适合表达「完成度 / 落在哪一档」 |
| **关联型**：流程、流向、层级等关系 | 漏斗图、和弦图、桑基图、矩形树图 | 漏斗看多阶段流程的关键流失环节；和弦 / 桑基看流向（注意始末分支宽度之和要相等，保持「能量平衡」）；矩形树图看层级 + 各节点占比 |
| **地理型**：和空间位置相关 | 地图类（点 / 区 / 热区等） | 把指标投影到地图上；价值取决于「投上去的指标」本身，而非图好不好看 |

> 提示：图表族之间会借用。比如趋势型的面积图、分布型的气泡图，有时也拿来做对比；同一张图的归类不必死板，关键看你要表达的关系。

### 按「分析目的」圈定优先图表

可视化方案要服务于分析目的。把目的性分析分成四类，各自有优先考虑的图表族（顺序灵活，可按具体问题改造）：

- **描述型分析**（说清业务现在长什么样）：优先**分布型、比例型、比较型**（涉及空间业务再加地理型）；想从时间 / 空间维度做更深的描述，可用**关联型、比例型**。
- **诊断型分析**（断因，找为什么）：图表选型多和诊断方法绑定。诊断常用多维分析、钻取、假设检验、交叉分析、相关分析，与之搭配的就是**比较型、分布型、关联型、趋势型**的组合。
- **预测型分析**（推断随自变量变化的走势）：以时间或某规模为自变量、看因变量趋势，**趋势型**是最常用选择。
- **指令型分析**（评估不同方案的可能效果，是分析的最终落脚点）：常用**比较型、关联型**，以**分布型、预测型**作辅助或备选。

### 一个中性示例

某出行平台要在月报里说清「新派单策略 B 的效果」：
- 先看整体表现（描述）→ 用柱状图比 A/B 两组完单率、用箱线图看接驾时长分布；
- 找差异来源（诊断）→ 用交叉分析配比较型图逐城对比，用散点 / 拟合看「接驾时长 vs 取消率」的相关；
- 展望推广后的走势（预测）→ 用折线图画完单率随时间的趋势；
- 给方案建议（指令）→ 用对比型图把「全量推广 vs 分城灰度」两套方案的预期效果并排放，辅以分布图说明风险。

### 易错点 / 边界

- **直方图 ≠ 柱状图**：柱状图比的是离散类别的频数，矩形**宽度无意义**、可留间隙；直方图表示连续值的频数与区间，矩形**宽度也有意义**（代表区间大小）、通常**连续排列**。频率直方图在自变量间距趋近无穷小时就过渡为概率密度图，两者常成对用（前者看真实离散分布，后者看连续拟合效果）。
- **一图塞太多维**：超过三维主信息就难读，宁可拆成多张子图，或用交互筛选。
- **为炫技选图**：图是为「让人一眼看懂关系」服务的，关系没选对，再漂亮也没用；热图、地图这类的价值取决于投上去的指标，而非视觉效果。
- **数据量 / 维度过大时**：要么先抽样保留重要特征、要么降维（如线性变换 PCA / LDA、非线性变换等）后在低维观察，要么用交互组件按需筛选，而不是硬把全量塞进一张静态图。
